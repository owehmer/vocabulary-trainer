# Projects

## Technologies
- Angular v. 21

## How we work
- Every task has a special .md (named after the task starting with an increasing number like 001-init-project.md) file which demonstrates the task and its requirements
- these .md files are stored in the ./tasks folder
- After a task you always commit these changes to git with a meaningful message (starting with the name of the .md file, max. 3 paragraphs and 100 characters) and include the plan .md file in the commit
  - e.g.: "001-init-project: Initialized the Angular project with the latest version. Set up the basic folder structure and installed necessary dependencies. This will serve as the foundation for our application development."
- We use that as a changelog, so whenever you want to know why a change was made you can look these files up

## Documentation
- Only write english in all files (excluding text for user displayed html stuff)

## Broad goal:

Ich möchte einen interaktiven Vokabeltrainer als webapp bauen. Ich spreche deutsch und möchte Schweizer Zürich deutsch lernen. Lass mich meine Vokabeln (Wort oder Satz) eingeben. diese werden gespeichert. Entweder nur auf deutsch und die App ergänz die Schweizer Seite, oder ich gebe beide Seiten ein. Beim Training soll es dann möglich sein sich einen Beispiel Satz aus dem echten Leben anzeigen zu lassen der diese Vokabel enthält (auf Schweizer deutsch mit deutscher Übersetzung). Optional: die Vokabel oder sogar der Satz kann von einer generieren Stimme abgespielt werden.
